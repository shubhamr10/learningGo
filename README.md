## Why Go and not PHP, nodeJS.

## Why not PHP/Laravel

* Compiles to a single binary file
* No runtimes to worry about.
* Statically typed, so no surprises at run time.
* Object-Oriented (sort of)
* Concurrency
* Cross-platform
* Excellent package management & testing built in.
